## 📖 Project Overview

The **Student Attendance Records** project is a simple webpage developed using **HTML5** and **CSS3** to display monthly attendance details of students in a tabular format.

This project demonstrates how different types of CSS (Inline, Internal, and External) can be applied to enhance the appearance of a webpage.

---

## 🛠️ Technologies Used

* **HTML5** – To create the structure of the webpage
* **CSS3** – To apply styling using:

  * Inline CSS
  * Internal CSS
  * External CSS

---

## ✨ Features

* Displays student attendance records in table format
* Use of Inline CSS for heading styling
* Use of Internal CSS for subheading and row styling
* Use of External CSS for overall table design
* Alternate row coloring using `nth-child` selector
* Status highlighting based on attendance performance
* Clean and structured layout

---

## 📚 Learning Objectives

Through this project, the following concepts are understood:

* Creating tables in HTML
* Styling webpages using:

  * Inline CSS
  * Internal CSS
  * External CSS
* Applying CSS selectors
* Using class-based styling
* Formatting table rows and headings

---

## 🚀 How to Run the Project

1. Save the HTML code in a file with `.html` extension (Example: `attendance.html`).
2. Create an external CSS file named `style.css`.
3. Link the CSS file to the HTML file.
4. Open the HTML file in any web browser.

---

## 📂 Project Components

* HTML Table
* Inline Styled Heading
* Internal CSS for formatting
* External CSS for table design
* Class-based status indicators

---

## 🎯 Output Description

The webpage shows a table containing student names, roll numbers, days present, and attendance status with proper styling applied using different CSS methods.

---

## 🙏 Conclusion

This project helps in understanding the implementation of various CSS styling techniques along with structured data representation using HTML tables.
